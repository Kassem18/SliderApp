## Troubleshooting

**Port in use?**

```bash
npm run dev -- --port 3000
```

**Clear cache?**

```bash
npx jest --clearCache
npm cache clean --force
npm install
```

**Build fails?**

```bash
npx tsc --noEmit  # Check TypeScript errors
npm run build     # Try again
```
